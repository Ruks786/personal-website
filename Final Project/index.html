html cheatsheet

HTML CODE

basic Tags description
<html> </html> Creates an HTML document
<head> </head> Sets off the title & other info that isn't displayed
<body> </body> Sets off the visible portion of the document
<title> </title> Puts name of the document in the title bar; when bookmarking pages, this is what is bookmarked
text tags description
<h1> </h1>----<h6> </h6> Creates headlines -- H1=largest, H6=smallest
<b> </b> Creates bold text (should use <strong> instead)
<i> </i> Creates italicized text (should use <em> instead)
<code> </code> Used to define source code, usually monospace
<cite> </cite> Creates a citation, usually processed in italics
<address> </address> Creates address section, usually processed in italics
<em> </em> Emphasizes a word
<strong> </strong> Emphasizes a word
<font size> </font> Sets size of font - 1 to 7
<font color> </font> Sets font color
<font face> </face> Defines the font used
links description
<a href="URL">clickable tex </a> Defines the font used
<a href="mailto:EMAIL_ADDRESS">clickable tex </a> Creates a hyperlink to an email address
Forms description
<form> </form> Defines a form
<select name=?> </select> Creates a pulldown menu
<option> </option> Sets off each menu item
<textarea name=? cols ="x" rows="y"> </textarea> Creates a text box area. Columns set the width; rows set the height
<input type="checkbox" name=? value=?> </input> Creates a checkbox.
<input type="radio" name=? value=?>> </input> Creates a radio button.
<input type="text" name=? size=?> </input> Creates a one-line text area. Size sets length, in characters.
<input type="submit" value=?> </input> Creates a submit button. Value sets the text in the submit button.
<input type="reset"> </input> Creates a reset button.
<input type="image" name=? src=? border=? alt=?> </input> Creates a submit button using an image.

HTML CODE

body attributes(used in news letters) description
<body bgcolor=?> Sets background color, using name or hex value
<body text=?> Sets text color, using name or hex value
<body link=?> Sets color of links, using name or hex value
formatting tags Description
<p> </p> Creates a new paragraph
<br> </br> Inserts a line break
<blockquote> </blockquote> Puts content in a quote - indents text from both sides
<div> </div> Used to format block content with CSS
<span> <span> Used to format inline content with CSS
lists tags Description
<ul> </ul> Creates an unordered list
<ol> </ol> Creates an ordered list
<li> </li> Encompasses each list item
<dl> </dl> Creates a definition list
<dt> </dt> Precedes each defintion term
<dd> </dd> Precedes each defintion
graphical elements Description
<hr> </hr> Inserts a horizontal line
<img src="URL" alt=?> Sets the alternate text for browsers that can't process images
HTML5 input attributes Description
<input type="email" name=?> Sets a single-line textbox for email addresses
<input type="url" name=?> Sets a single-line textbox for URLs
<input type="number" name=?> Sets a single-line textbox for number
<input type="range" name=?> Sets a single-line textbox for range of number
<input type="date/month/week/time" name=?> Sets a single-line text box with a calendar showing the date/month/week/time
<input type="search" name=?> Sets a single-line textbox for searching
<input type="color" name=?> Sets a single-line textbox for picking color
tables description
<table> </table> Creates a table
<tr> </tr> Sets off each row in a table
<td> </td> Sets off each cell in a row
<th> </th> Sets off the table header (a normal cell with bold, centered text)
Table attributes Description
<table border=?> </table> Sets the width of the border around table cells
<table cellspacing> </table> Sets amount of space between table cells
<tr align=?> </tr> Sets alignment for cells within the row (left/center/right)
<td align=?> </td> Sets alignment for cells (left/center/right)
<td rowspan=?> </td> Sets number of rows a cell should span (default=1)
<td colspan=?> </td> Sets number of columns a cell should span
